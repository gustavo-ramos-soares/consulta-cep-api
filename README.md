# Consulta CEP API

Aplicação web desenvolvida com HTML, CSS e JavaScript para consulta de endereços através da API ViaCEP.

## Funcionalidades

* Consulta de CEP em tempo real
* Preenchimento automático de endereço
* Exibição de logradouro, bairro, cidade e estado
* Tratamento de CEP inválido ou inexistente
* Tratamento de erros de conexão

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript ES6+
* Fetch API
* Async/Await

## API Utilizada

ViaCEP

https://viacep.com.br

## Aprendizados

Durante o desenvolvimento foram praticados conceitos de:

* Manipulação do DOM
* Requisições HTTP
* Consumo de APIs REST
* Programação assíncrona com Async/Await
* Tratamento de erros com Try/Catch
